# Changelog

## 5.2.3

- Put a little email envelope icon (Unicode, so no additional assets are loaded) on a player card to make it it easier to email registrants. Opens a `mailto` link addressed to the player with the name of the event in the subject.

## 5.2.2

- Added an "Edit registration details" button to each player's card, linking to their ETECF registration form. 
- USCF IDs now link to `ratings.uschess.org`.
- Added a setting to show / hide the registration count badge (on by default), with some help text about page caching.
- Player cards set `DONOTCACHEPAGE`, so a page cache can't store and serve it to anonymous visitors.

## 5.2.1

- Updating README "Versions" section with known versions of ETECF that ETR is known to work with.

## 5.2

- First release.
